HTML

  • The <base> element defines the base URL for a webpage.
  • The <body> element represents the visible content shown to the user.
  • The <br> element creates a line of empty space, or a line break, between two blocks of content.
  • The <charset> element sets our character encoding.
  • The <h1>, <h2>, <h3>, <h4>, <h5> and <h6> elements represent the level of heading a given text block represents.
  • The <head> element contains information about the webpage.
  • The <http-equiv> element sets an HTTP header for content.
  • The <img> element contains information about images that are displayed on the webpage.
  • The <link> element connects an external resource to the HTML document.
  • The <meta> element defines metadata such as the character set, description, keywords, author, and viewport
  • The <p> element represents a paragraph or block of text
  • The <script> element embeds client-side scripts in the HTML document.
  • The <section> element defines a section in a document that contains thematic content.
  • The <style> element defines the CSS styles associated with a website.
  • The <title> element defines the title for a webpage. This element is required and is extremely important for Search Engine Optimization (SEO).
  • The <ul>, <ol> and <li> elements represent unordered lists, ordered lists and list items.
  • The <viewport> element sets the default view for our webpage.

CSS

  • Do Not Repeat Yourself (DRY) is a coding principle that encourages developers to reuse or share pieces of code in order to reduce the number of lines of code that need to be written or maintained.
  • A border assigns a border around the selector.
  • A box-shadow CSS property adds shadow effects around an element's frame.
  • A card is a class selector and will cause any HTML element assigned to the class to gain the "Card" property assignments.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Javascript

  • A control flow is the order in which a computer executes code in a script.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • A function is a reusable block of code that contain instructions to perform a specific task.
  • A variable is a named container that allows us to store data in our code.
  • An Array is a single variable that is used to hold a group of data.
  • An if statement is a way of introducing decision-making into our code and will only execute if the statement meets a condition, or is true.
